About Grivet Support Services

A collection of Spring Cloud, Spring Boot Admin services powered by the Netflix OSS stack

Project Modules

This project has declared the following modules:

Name Description
API Gateway This API Gateway is a single point of entry (and control) for front-end clients, which could be browser based or mobile. The client only has to know the URL of one server, and the back-end can be refactored at will with no change, which is a significant advantage. There are other advantages in terms of centralization and control: rate limiting, authentication, auditing and logging.
Spring Boot Application Admin Provides an overview of all running Spring Boot Micro Services in Grivet environment.
Configuration Server A central place to manage external properties for applications across all environments.
Spring Boot Application Dashboard Provides visual monitoring for all running Spring Boot Micro Services in Grivet environment.
Discovery Service This discovery service is powered by Eureka. Eureka is a REST based service that is primarily used in the AWS cloud for locating services for the purpose of load balancing and fail-over of middle-tier servers.
